## 结构说明

### 主容器
tag: `.project-main`
note: 页面主体容器，横屏模式布局，包含头部、流程图、标签页等核心区域

### 页面头部
tag: `.main-header`
note: 页面顶部导航栏，分左中右三栏，展示项目基本信息和快捷操作入口

### 头部左侧
tag: `.main-header-left`
note: 展示项目核心信息和状态，包含关注、项目名、状态和操作菜单

### 关注图标
tag: `.follower-icon`
note: 点击切换项目关注状态，已关注时图标变蓝

### 项目名称
tag: `.header-title`
note: 显示项目名称，点击可编辑

### 项目状态
tag: `.header-progress`
note: 显示项目当前状态（进行中、已完成、变更中、暂停等），不同状态对应不同颜色

### 操作菜单
tag: `.operation-btn .suspension`
note: 下拉菜单，包含取消立项/继续/暂停、变更、删除等项目管理操作

### 头部中间
tag: `.main-header-middle`
note: 快捷操作按钮区，包含返回首页等常用功能

### 头部右侧
tag: `.main-header-right`
note: 用户操作区，包含时间线、角色切换和用户头像

### 时间线按钮
tag: `.header-action-item`
note: 点击打开时间线独立弹窗，查看和编辑项目节点时间线

### 角色切换按钮
tag: `.header-action-item`
note: 点击打开角色独立弹窗，批量设置节点负责人

### 用户头像下拉
tag: `.avatar-img .role-list`
note: 显示当前用户头像，点击展开角色列表，可切换当前角色

### 流程图区域
tag: `.main-chart`
note: 核心功能区，展示项目节点流程图和节点详情

### 节点流程图
tag: `flowChart` 组件
note: 使用 Canvas 渲染的项目节点流程图，支持节点点击、悬停交互，展示节点层级和状态关系

### 节点详情侧边栏
tag: `.main-node-details`
note: 流程图右侧的节点详情面板，展示选中节点的详细信息（交付物、审批、配置等）

### 标签页内容区
tag: `.main-article`
note: 项目详细信息展示区，通过标签页切换不同模块内容

### 标签页导航栏
tag: `.full-border .tab-container`
note: 固定在顶部的标签页导航，支持吸顶效果

### 标签页 Tab 列表
tag: `.list-tabs .tab .tab-name`
note: 包含基本信息、文件管理、需求、缺陷、结算、评价、操作记录等 Tab，点击切换内容

### 右侧操作按钮
tag: `.tabs-right-btn`
note: 标签页右侧的操作按钮，包含刷新按钮和新增需求按钮（需求 Tab 下）

### 标签页内容组件
tag: `.tab-content`
note: 动态组件区域，根据当前 Tab 加载对应组件内容

### 基本信息
tag: `basicInfo` 组件
note: 展示项目名称、描述、客户、机型、难度系数、创建时间、交付日期等基础信息

### 文件管理
tag: `deliveryFile` 组件
note: 项目文件管理模块，支持文件上传、下载、冻结等操作

### 需求管理
tag: `requirement` 组件
note: 项目需求列表，支持新增、编辑、删除需求

### 缺陷管理
tag: `defect` 组件
note: 项目缺陷列表，支持缺陷跟踪和管理

### 项目结算
tag: `projectSettlement` 组件
note: 项目结算信息，包含贡献度、变更日志、积分计算等

### 项目评价
tag: `projectEvaluate` 组件
note: 项目评价表单，支持对项目进行多维评价

### 操作记录
tag: `records` 组件
note: 项目操作历史记录，按时间倒序展示所有操作日志

### 删除状态展示
tag: `.delete-show`
note: 项目已删除时的提示页面，显示空状态和返回首页按钮

### 进度历史抽屉
tag: `ly-drawer`
note: 右侧抽屉式弹窗，展示项目节点的进展历史记录

### 项目变更弹窗
tag: `changes-pop`
note: 项目变更流程弹窗，支持选择变更范围、原因和负责人

### 项目评价弹窗
tag: `project-evaluate-pop`
note: 项目评价表单弹窗，包含多个评价维度的打分和填写

### 时间线独立弹窗
tag: `refTimelinePop .ly-popup-dialog .timeline-only`
note: 时间线编辑弹窗，横向展示所有节点的时间关系，支持点击节点设置时间范围

### 角色独立弹窗
tag: `refRolePop .ly-popup-dialog .role-only`
note: 角色批量设置弹窗，按角色分组展示节点，支持批量设置节点负责人

### 时间线设置弹窗
tag: `refChangeTimelinePop .timeline_setting_wrapper`
note: 单个节点时间范围设置弹窗，使用日期范围选择器设置节点开始和结束时间

### 节点负责人变更弹窗
tag: `refChangeNodeOwnerPop .node-owner-main`
note: 批量变更节点负责人弹窗，显示已有负责人的节点冲突，支持全部或部分更换

### 暂停/继续项目弹窗
tag: `pauseProjectPopup .alter-main`
note: 暂停或继续项目的原因填写弹窗，必填原因字段

### 取消立项弹窗
tag: `cancelProjectPopup .alter-main`
note: 取消项目立项的原因填写弹窗，原因可选填
