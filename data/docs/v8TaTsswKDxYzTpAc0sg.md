# 项目概述

## 项目名称

Duolingo 自动化管理平台 - H5 前端

## 项目定位

这是一个基于 Vue 3 和 Vant UI 组件构建的移动端 Web 应用程序。用于在移动设备上管理 Duolingo 账户、客户和自动化任务。

## 技术定位

- **前端类型**: H5 移动端 Web 应用
- **目标平台**: iOS Safari 和 Android Chrome
- **设计基准**: 375px (iPhone SE)

## 项目关系

这是 H5 前端项目。相关项目：

- **后端 Django API**: 位于 `../Duolingo_backend/`
- **PC 前端**: 位于 `../Duolingo_frontend/` (基于 Element Plus 的管理面板)
- **PRD 文档**: 位于 `./PRD.md` (完整产品需求文档)

## 核心功能

1. **用户认证**: 登录、注册、密码管理
2. **客户管理**: 客户信息维护
3. **账户管理**: Duolingo 账户验证和管理
4. **订单管理**: 订单创建、执行和监控
5. **任务执行**: 异步任务调度和执行

## 业务模块

应用按后端模块组织：

- **SM (系统管理)**: 登录、注册、仪表盘
- **BDM (业务数据管理)**: 客户、Duolingo 账户
- **APS (异步处理系统)**: 订单、任务执行
