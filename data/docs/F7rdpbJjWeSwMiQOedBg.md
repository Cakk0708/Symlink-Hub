# 技术栈

## 核心框架

- **框架**: Vue 3 组合式 API (`<script setup>`)
- **构建工具**: Vite 5.0+
- **UI 库**: Vant 4.8+ (移动端组件库)
- **状态管理**: Pinia 2.1+
- **路由**: Vue Router 4.2+
- **HTTP 客户端**: Axios 配合自定义请求封装

## 开发语言

- **JavaScript**: ES6+ 语法
- **CSS**: SCSS 预处理

## 目标平台

- **设备**: H5 移动端 Web
- **设计基准**: 375px (iPhone SE)
- **目标浏览器**: iOS Safari 和 Android Chrome

## 依赖管理

- **包管理器**: npm
- **配置文件**: package.json
