# Dashboard 模块说明

## 模块定位

- **路径**: `/dashboard`
- **别名**: 仪表盘/首页/项目首页
- **所属**: 系统核心模块

## 功能概述

Dashboard 是系统的首页，展示关键业务统计数据和可视化图表，为用户提供项目整体运营状况的概览。

## 核心功能

### 1. 统计卡片
- **总用户数**: 展示系统中的用户总数
- **昨日订单**: 展示昨日订单完成数量
- **今日订单**: 展示今日订单完成数量
- **营业额**: 展示今日营业额统计

### 2. 图表展示

#### 今日订单趋势图
- **类型**: 可切换的折线图/柱状图
- **数据**: 按小时展示今日订单数量变化
- **刷新**: 每分钟自动更新数据
- **交互**: 支持图表类型切换

#### 订单分类分布图
- **类型**: 环形饼图
- **数据**: 展示不同类型订单的数量分布
- **类型**: EXP、GEMS、3X_XP、SIGN
- **交互**: 鼠标悬停显示详细数据

## 技术实现

### 数据获取
- 使用 `request.js` 工具发起请求
- 接口: `GET /SM/dashboard`
- 自动刷新: 使用 `setInterval` 每分钟更新
- 错误处理: 统一的错误提示

### 组件结构
```vue
<template>
  <div class="dashboard">
    <!-- 统计卡片 -->
    <el-row :gutter="20">
      <el-col v-for="card in statCards" :key="card.title" :span="6">
        <stat-card :card="card" />
      </el-col>
    </el-row>

    <!-- 图表区域 -->
    <el-row :gutter="20" class="charts-row">
      <el-col :span="16">
        <trend-chart />
      </el-col>
      <el-col :span="8">
        <distribution-chart />
      </el-col>
    </el-row>
  </div>
</template>
```

## 交互设计

### 自动刷新
- 组件挂载后启动定时器
- 每分钟调用一次数据接口
- 组件卸载时清除定时器
- 首次加载立即获取数据

### 图表切换
- 提供切换按钮（折线图/柱状图）
- 使用 ECharts 配置实现
- 保持数据不变，切换展示方式

## 开发注意事项

### 性能优化
- 图表数据量大时考虑虚拟渲染
- 定时器管理避免内存泄漏
- 数据加载时显示 loading 状态

### 响应式设计
- 统计卡片使用 Grid 布局
- 图表区域自适应宽度
- 移动端布局调整

### 代码规范
- 使用 Composition API
- 提取图表组件为独立组件
- 统一错误处理机制
