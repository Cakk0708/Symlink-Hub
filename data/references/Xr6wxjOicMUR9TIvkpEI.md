# 布局文档模板参考

> 按需读取此文件。在第四步生成布局文档时加载。

---

## 完整文档模板

````markdown
# {页面名称} 布局说明

> 最后更新：{YYYY-MM-DD}
> 页面文件：`{page_path}`

---

## 页面结构

{按层级缩进列出所有区块，子区块缩进 4 个空格}

示例：
- 标题
- 搜索栏
- 功能块
    - 创建项目
    - 报表
    - 后台
- 项目列表
    - 项目分类
    - 项目筛选
    - 项目列表

---

## 结构说明

{为每个区块生成一个小节，按页面结构顺序排列}

### {区块名称}
tag: {CSS class 或 id，格式为 .class-name 或 #id-name，多个用空格分隔；若无语义 tag 则留空}
note: {该区块的功能说明，描述它是什么、做什么、关键交互}

{子区块直接跟在父区块后，无需缩进，通过顺序和命名体现层级}

### {子区块名称}
tag: {.class-name}
note: {功能说明}
````

---

## tag 填写规则

**优先级（从高到低）：**
1. 语义化 class：`.main-search`、`.project-list`、`.feature-block`
2. 有意义的 id：`#search-bar`、`#main-nav`
3. 组件名转换：`<ProjectCard>` → `ProjectCard 组件`
4. 语义 HTML 标签：`header`、`nav`、`main`、`footer`、`aside`
5. 动态 class 中的静态部分：`:class="['btn', isActive && 'btn-active']"` → `.btn`
6. 无法确定时留空，在第六步标记为"需人工确认"

**多个 tag 时空格分隔：**
```
tag: .search-bar .search-input
```

---

## note 填写规则

note 应描述：
- **是什么**：区块的身份（标题、搜索栏、列表等）
- **做什么**：核心功能（输入关键词搜索项目、点击跳转详情等）
- **关键交互**（可选）：点击、输入、筛选、展开等

**不需要写**：样式描述、颜色、字体大小等视觉细节。

**示例：**
```
note: 页面主标语，展示系统定位
note: 项目搜索入口，支持按项目名称模糊搜索，含清空按钮
note: 功能导航区，包含三个快捷入口卡片
note: 点击进入数据报表页，查看项目完成度统计
note: 项目列表主体区域，含分类 Tab、筛选条件和数据表格
```

---

## 页面结构书写规则

- 使用 `- ` 无序列表，子级缩进 **4 个空格**
- 只列出有实际内容或交互意义的区块，忽略纯样式包裹层
- 重复渲染的列表项（`v-for`）合并为一个条目，如"项目列表项"
- 弹窗、抽屉等浮层可列在末尾，标注 `（浮层）`

**示例：**
```markdown
- 顶部导航
- 页面标题
- 搜索栏
- 功能卡片区
    - 项目入口
    - 报表入口
    - 后台入口
- 项目列表
    - 分类 Tab
    - 筛选栏
        - 类型筛选
        - 负责人筛选
        - 客户筛选
        - 日期筛选
    - 数据表格
        - 表头
        - 列表行
- 新建项目弹窗（浮层）
```

## 完整示例文档

````markdown
# 首页 布局说明

> 最后更新：2026-03-26
> 页面文件：`src/views/home/index.vue`

---

## 页面结构

- 标题
- 搜索栏
- 功能块
    - 创建项目
    - 报表
    - 后台
- 项目列表
    - 项目分类
    - 项目筛选
        - 类型
        - 负责人
        - 客户
        - 下单
        - 日期范围
    - 项目表格

---

## 结构说明

### 标题
tag: .main-slogan
note: 页面主标语，展示系统定位

### 搜索栏
tag: .main-search
note: 项目全局搜索入口，支持按项目编号或名称模糊搜索，含清空按钮

### 功能块
tag: .feature-block
note: 三个快捷功能入口，横向排列展示

### 创建项目
tag: .feature-item .feature-project
note: 点击打开新建项目流程，跳转项目创建页

### 报表
tag: .feature-item .feature-report
note: 点击进入项目报表页，查看完成度与进度统计

### 后台
tag: .feature-item .feature-admin
note: 点击进入系统管理后台，含权限配置与参数管理

### 项目列表
tag: .project-list
note: 页面核心区域，展示当前用户可见的项目数据

### 项目分类
tag: .project-tabs
note: 项目状态分类 Tab，包含全部、进行中、变更中、变更完成、已完成等状态

### 项目筛选
tag: .project-filter
note: 多维度筛选条件区，筛选结果实时更新列表

### 类型
tag: .filter-type
note: 按项目类型筛选，下拉单选

### 负责人
tag: .filter-owner
note: 按负责人筛选，下拉单选

### 客户
tag: .filter-customer
note: 按客户筛选，下拉单选

### 下单
tag: .filter-order
note: 按下单状态筛选，下拉单选

### 日期范围
tag: .filter-date
note: 开始日期与结束日期范围选择，日期区间筛选

### 项目表格
tag: .project-table
note: 项目数据表格，含项目标题、变更类型、变更原因、负责人、创建时间、变更完成时间列，支持排序
````
