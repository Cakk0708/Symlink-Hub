# 存储键名

> 项目中使用的本地存储键名规范

| 键名 | 说明 | 位置 |
|------|------|------|
| `betaToken` | Token 存储 | localStorage + uni.storage |
| `betaUserInfo` | 用户信息 | localStorage + uni.storage |
| `kNodeList` | 节点列表 | uni.storage |
| `proOwner` | 项目负责人 | uni.storage |
| `creator` | 创建者 | uni.storage |
| `kModel` | 机型名称 | uni.storage |
| `conciseEnum` | 简化枚举 | uni.storage |
