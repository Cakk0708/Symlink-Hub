# 开发命令

## 安装依赖

```bash
npm install
```

## 开发服务器

```bash
npm run dev
```

启动本地开发服务器，默认运行在 `http://localhost:5173`

## 生产构建

```bash
npm run build
```

构建生产版本，输出到 `dist/` 目录

## 预览生产构建

```bash
npm run preview
```

在本地预览生产构建结果

## 代码检查

建议添加以下命令（目前未配置）：

```bash
npm run lint      # ESLint 代码检查
npm run test:unit # 单元测试
npm run test:e2e  # E2E 测试
```
