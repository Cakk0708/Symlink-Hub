## 结构说明

### 页面容器
tag: .order-detail-container
note: 详情页顶层容器，浅灰背景，纵向排列所有信息卡片

### 顶部导航栏
tag: .page-header
note: 页面顶部白色横条，左侧返回按钮，右侧"订单详情"标题

### 返回按钮
tag:
note: 带箭头图标的"返回"按钮，点击返回订单列表页

### 页面标题
tag: .page-title
note: 显示"订单详情"文字标题

### 订单信息卡片
tag: .info-card
note: 展示订单基础信息的卡片，加载时显示loading状态

### 卡片头部
tag: .card-header
note: 卡片标题栏，显示"订单信息"

### 订单描述列表
tag: el-descriptions
note: 两列带边框的描述列表，展示订单ID、编号、类型、业务类型、状态、价格、计划时间、版本信息

### 订单ID
tag:
note: 显示订单自增ID

### 订单编号
tag:
note: 显示订单唯一编号

### 订单类型
tag:
note: 以标签形式显示订单类型（EXP/GEMS/3X_XP/SIGN），不同类型对应不同颜色

### 业务类型
tag:
note: 显示业务类型文字（付费/免费）

### 订单状态
tag:
note: 以标签形式显示订单当前状态，不同状态对应不同颜色

### 价格
tag:
note: 显示订单价格，格式为"¥"加金额，无价格时显示"-"

### 计划时间
tag:
note: 显示订单计划执行时间

### 版本信息
tag:
note: 当订单存在版本号时条件显示，格式为"v"加版本号

### 订单操作卡片
tag: .actions-card
note: 订单状态操作按钮区，根据当前订单状态自动启用/禁用对应按钮

### 操作按钮组
tag: .detail-actions
note: 横向排列的操作按钮集合，支持自动换行

### 开始执行按钮
tag:
note: 将订单状态从waiting改为doing，当订单已在执行中或已完成时禁用

### 暂停按钮
tag:
note: 将订单状态从doing改为pause，点击弹出确认对话框，仅进行中状态可操作

### 恢复执行按钮
tag:
note: 将订单状态从pause改为doing，点击弹出确认对话框，仅暂停状态可操作

### 标记完成按钮
tag:
note: 将订单状态改为done，点击弹出确认对话框，已完成或已取消状态时禁用

### 重置为等待按钮
tag:
note: 将订单状态重置为waiting，点击弹出确认对话框，等待或暂停状态时禁用

### 取消订单按钮
tag:
note: 将订单状态改为cancel，点击弹出确认对话框，已完成或已取消状态时禁用

### 重新开始按钮
tag:
note: 仅在订单状态为cancel时条件显示，点击弹出确认对话框，将状态恢复为waiting

### 删除订单按钮
tag:
note: 仅在订单状态为cancel时条件显示，点击弹出确认对话框，确认后删除订单并返回列表页

### 执行进度卡片
tag: .progress-card
note: 展示订单执行进度数据的卡片

### 进度描述列表
tag: el-descriptions
note: 两列带边框的描述列表，展示计划数量、初始值、已完成数量、完成百分比、计划时长、进度条

### 计划数量
tag:
note: 显示订单的计划执行总量

### 初始值
tag:
note: 显示订单执行前的初始值

### 已完成数量
tag:
note: 显示当前已完成的数量

### 完成百分比
tag:
note: 根据已完成数量/计划数量自动计算的百分比数值

### 计划时长
tag:
note: 当存在计划时长时条件显示，单位为分钟

### 进度条
tag:
note: Element Plus进度条组件，宽度占满两列，颜色根据订单状态变化（完成-绿/错误或取消-红/其他-蓝）

### 类型特定信息卡片
tag: .type-detail-card
note: 当订单存在typeDetail数据时条件显示，动态渲染该订单类型的专属字段

### 动态字段列表
tag: el-descriptions
note: 遍历typeDetail对象的所有键值对，动态生成描述列表项

### 关联信息卡片
tag: .relations-card
note: 展示订单关联的账户、客户、组织信息的卡片

### 关联描述列表
tag: el-descriptions
note: 单列带边框的描述列表，展示账户、客户、组织三组关联信息

### 账户信息
tag:
note: 显示关联Duolingo账户的昵称、用户名、手机、邮箱、备注

### 客户信息
tag:
note: 显示关联客户的名称和编码

### 组织信息
tag:
note: 显示关联组织的名称和编码

### 时间信息卡片
tag: .time-card
note: 展示订单时间戳的卡片

### 时间描述列表
tag: el-descriptions
note: 两列带边框的描述列表，展示创建时间和更新时间
