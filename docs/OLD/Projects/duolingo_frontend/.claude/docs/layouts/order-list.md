## 结构说明

### 页面容器
tag: .order-container
note: 页面顶层容器，承载订单列表主体内容和新增订单弹窗

### 列表卡片
tag: el-card
note: 订单数据展示的主卡片区域，包含搜索、表格、分页三部分

### 卡片头部
tag: .card-header
note: 卡片顶部标题栏，左侧显示"订单管理"标题，右侧放置新增订单入口

### 标题
tag:
note: 显示"订单管理"文字标题，标识当前页面功能

### 新增按钮
tag:
note: "新增订单"按钮，点击打开新增订单弹窗

### 搜索栏
tag: .search-form
note: 行内表单搜索区，支持按订单编号、客户ID、订单状态、订单类型进行筛选，含搜索和重置操作

### 订单编号输入
tag:
note: 订单编号文本输入框，支持模糊搜索

### 客户ID输入
tag:
note: 客户ID文本输入框，按客户标识筛选

### 订单状态筛选
tag:
note: 订单状态下拉选择器，选项从后端枚举接口获取，包含等待/进行中/完成/取消/错误/暂停等状态

### 订单类型筛选
tag:
note: 订单类型下拉选择器，选项从后端枚举接口获取，包含经验/宝石/三倍经验/签到等类型

### 搜索按钮
tag:
note: 触发搜索，重新从第一页加载订单列表数据

### 重置按钮
tag:
note: 清空所有搜索条件并重新加载数据

### 订单表格
tag: el-table
note: 订单数据表格，双击行可跳转到对应订单详情页，高度自适应视口

### ID列
tag:
note: 显示订单自增ID，宽度60px

### 订单编号列
tag:
note: 显示订单编号，渲染为可点击链接，点击跳转订单详情页

### 订单类型列
tag:
note: 以标签（el-tag）形式显示订单类型，不同类型对应不同颜色（EXP-蓝/GEMS-绿/3X_XP-橙/SIGN-灰）

### 业务类型列
tag:
note: 显示业务类型文字，付费或免费

### 客户名称列
tag:
note: 显示关联客户的名称

### 账号昵称列
tag:
note: 显示关联Duolingo账号的昵称

### 进度列
tag:
note: 显示订单执行进度，格式为"已完成数量/计划数量"

### 订单状态列
tag:
note: 以标签（el-tag）形式显示订单状态，不同状态对应不同颜色（等待-灰/进行中-蓝/完成-绿/取消-橙/错误-红/暂停-橙）

### 计划时间列
tag:
note: 显示订单计划执行时间

### 创建时间列
tag:
note: 显示订单创建时间

### 操作列
tag: .operation-buttons
note: 行操作按钮区，固定在表格右侧，包含详情和删除两个操作

### 详情按钮
tag:
note: 点击跳转到对应订单的详情页

### 删除按钮
tag:
note: 点击弹出确认对话框，确认后删除对应订单并刷新列表

### 分页
tag: .container-pagination
note: 分页控件，靠右对齐，支持上一页/页码/下一页切换

### 新增订单弹窗
tag: el-dialog
note: 新增订单的模态对话框，宽度470px，包含订单模式切换、客户/账户选择、类型特定字段配置

### 订单模式选择
tag:
note: 单选组，切换"自定义订单"和"选择套餐"两种下单模式

### 套餐分类选择
tag:
note: 套餐模式下显示，下拉选择套餐分类（经验/宝石/三倍经验/签到），选择后自动加载对应分类的套餐列表

### 套餐选择
tag:
note: 套餐模式下显示，下拉选择具体套餐，选择后自动填充订单类型、价格、计划数量等字段

### 客户选择
tag:
note: 下拉选择客户，必填项，选择后自动加载该客户下的账户列表

### 账户选择
tag:
note: 下拉选择账户，必填项，需先选择客户后才可操作，支持搜索过滤，仅显示未禁用的账户

### 订单类型选择
tag:
note: 单选组选择订单类型（EXP/GEMS/三倍经验/签到），套餐模式下禁用自动填充

### 业务类型选择
tag:
note: 单选组选择业务类型（付费/免费），套餐模式下禁用自动填充

### 价格输入
tag:
note: 数字输入框，设置订单价格，保留两位小数，套餐模式下禁用自动填充

### EXP类型字段
tag:
note: 当订单类型为EXP时条件显示的字段组，包含计划数量、计划周期（分钟）、执行时间选择

### GEMS类型字段
tag:
note: 当订单类型为GEMS时条件显示的字段组，包含计划数量、执行时间选择

### 3x_xp类型字段
tag:
note: 当订单类型为三倍经验时条件显示的字段组，包含计划数量、执行时间选择

### SIGN类型字段
tag:
note: 当订单类型为签到时条件显示的字段组，包含计划数量、开始日期、结束日期、每日执行时间

### 弹窗底部
tag: .dialog-footer
note: 弹窗底部操作区，包含取消和确定按钮
