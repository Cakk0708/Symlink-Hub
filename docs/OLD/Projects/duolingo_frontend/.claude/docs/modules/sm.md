# SM (系统管理) 模块说明

## 模块定位

- **路径**: `/sm`
- **别名**: 系统管理
- **所属**: 核心业务模块

## 功能概述

SM (System Management) 模块是系统的核心管理模块，负责系统的用户管理、权限管理等基础功能。当前实现主要包含用户管理功能。

## 核心功能

### 1. 用户管理
- **用户列表**: 展示系统所有用户信息
- **新增用户**: 添加新用户到系统
- **用户信息**: 用户名、真实姓名、所属组织、创建时间
- **分页功能**: 支持大量数据的分页展示

### 2. 权限管理
- **角色分配**: 用户角色管理（预留）
- **权限控制**: 基于角色的访问控制（预留）

## 技术实现

### 路由结构
```javascript
{
  path: '/sm',
  component: Layout,
  children: [
    {
      path: 'user',
      name: 'UserManagement',
      component: () => import('@/views/SM/user/index.vue'),
      meta: { title: '用户管理', icon: 'User' }
    }
  ]
}
```

### 数据接口
- `GET /SM/user` - 获取用户列表
- `POST /SM/user` - 创建新用户
- `DELETE /SM/user` - 删除用户（预留）
- `PUT /SM/user` - 更新用户信息（预留）

### 组件结构
```
SM/
├── index.vue          # SM 模块主页
└── user/
    └── index.vue      # 用户管理页面
```

## 开发指南

### 添加新子模块
1. 在 `views/SM/` 下创建新模块目录
2. 在 `router/index.js` 中添加路由配置
3. 在 Layout 中会自动生成菜单
4. 创建对应的模块说明文档

### 权限设计
当前系统采用简单的 token 认证，后续可扩展：
- RBAC（基于角色的访问控制）
- 菜单权限控制
- 操作权限控制
