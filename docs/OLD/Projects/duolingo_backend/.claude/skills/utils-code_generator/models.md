# Code Generator 数据模型说明

## 当前实现 (v1.0)

当前版本不使用独立的数据模型，而是直接在调用方模型上进行查询统计。

### 依赖的模型字段要求

调用 `generate_code` 函数的模型类必须包含以下字段：

```python
class SomeModel(models.Model):
    code = models.CharField(
        max_length=50,
        unique=True,
        verbose_name='编码',
        db_index=True  # 建议添加索引以提升查询性能
    )
```

### 字段规格

| 属性 | 要求 | 说明 |
|------|------|------|
| 字段名 | `code` | 固定名称，函数内部硬编码 |
| 类型 | `CharField` | 字符串类型 |
| 长度 | 建议 ≥ 20 | 取决于前缀、日期格式、序号位数的总长度 |
| 约束 | `unique=True` | 必须唯一，防止重复编码 |
| 索引 | 建议添加 | 提升前缀查询性能 |

## Phase 3: CodeSequence 模型 (v2.5)

### 模型定义

当演进到 Phase 3 时，将引入独立的序列管理模型：

```python
from django.db import models

class CodeSequence(models.Model):
    """编码序列管理表

    用于管理各业务模型的编码序列，支持：
    - 按前缀和日期分组管理序列
    - 序列状态可追溯
    - 支持手动调整序列值
    """
    prefix = models.CharField(
        max_length=10,
        verbose_name='前缀',
        db_index=True,
        help_text='业务编码前缀，如 C、O、P 等'
    )
    date = models.DateField(
        verbose_name='日期',
        db_index=True,
        help_text='序列所属日期'
    )
    count = models.IntegerField(
        default=0,
        verbose_name='当前序号',
        help_text='当前已使用的序列数'
    )
    created_at = models.DateTimeField(
        auto_now_add=True,
        verbose_name='创建时间'
    )
    updated_at = models.DateTimeField(
        auto_now=True,
        verbose_name='更新时间'
    )

    class Meta:
        db_table = 'utils_code_sequence'
        verbose_name = '编码序列'
        verbose_name_plural = '编码序列'
        unique_together = [['prefix', 'date']]
        ordering = ['-date', '-prefix']

    def __str__(self):
        return f'{self.prefix}{self.date.strftime("%Y%m%d")}: {self.count}'

    @classmethod
    def get_next_sequence(cls, prefix: str, date) -> int:
        """获取下一个序列号（线程安全）

        Args:
            prefix: 编码前缀
            date: 日期对象

        Returns:
            下一个可用的序列号
        """
        from django.db import transaction

        with transaction.atomic():
            obj, created = cls.objects.select_for_update().get_or_create(
                prefix=prefix,
                date=date,
                defaults={'count': 1}
            )
            if not created:
                obj.count += 1
                obj.save(update_fields=['count'])
            return obj.count

    @classmethod
    def reset_sequence(cls, prefix: str, date, new_count: int = 0):
        """重置序列号（谨慎使用）

        Args:
            prefix: 编码前缀
            date: 日期对象
            new_count: 新的序列值
        """
        obj, _ = cls.objects.update_or_create(
            prefix=prefix,
            date=date,
            defaults={'count': new_count}
        )
        return obj

    @classmethod
    def get_current_sequence(cls, prefix: str, date) -> int:
        """获取当前序列号（不递增）

        Args:
            prefix: 编码前缀
            date: 日期对象

        Returns:
            当前序列号，不存在返回 0
        """
        obj = cls.objects.filter(prefix=prefix, date=date).first()
        return obj.count if obj else 0
```

### 历史数据初始化

```python
from django.db import migrations

def initialize_sequences(apps, schema_editor):
    """从现有数据初始化序列表"""
    Customer = apps.get_model('BDM', 'Customer')
    CodeSequence = apps.get_model('utils', 'CodeSequence')

    from datetime import datetime
    from collections import defaultdict

    # 统计现有编码
    sequences = defaultdict(int)
    for customer in Customer.objects.all():
        if customer.code and len(customer.code) >= 9:
            # 假设格式为 CYYYYMMDDXXX
            prefix = customer.code[0]
            date_str = customer.code[1:9]
            try:
                date = datetime.strptime(date_str, '%Y%m%d').date()
                sequences[(prefix, date)] += 1
            except ValueError:
                continue

    # 批量创建序列记录
    seq_objects = [
        CodeSequence(prefix=p, date=d, count=count)
        for (p, d), count in sequences.items()
    ]
    CodeSequence.objects.bulk_create(seq_objects)

class Migration(migrations.Migration):

    dependencies = [
        ('utils', '0002_create_codesequence'),
    ]

    operations = [
        migrations.RunPython(initialize_sequences),
    ]
```

## Admin 配置

```python
from django.contrib import admin
from .models import CodeSequence

@admin.register(CodeSequence)
class CodeSequenceAdmin(admin.ModelAdmin):
    list_display = ['prefix', 'date', 'count', 'updated_at']
    list_filter = ['prefix', 'date']
    search_fields = ['prefix']
    readonly_fields = ['created_at', 'updated_at']

    def has_add_permission(self, request):
        # 禁用手动添加，通过系统自动创建
        return False

    def has_delete_permission(self, request, obj=None):
        # 禁止删除，保护历史记录
        return False
```
